# 📄 Обновление: Многостраничная навигация

## Изменения

### Было:
- Одна длинная страница со скроллингом
- Все секции видны одновременно
- Нужно скроллить для навигации
- Intersection Observer для автоактивации вкладок

### Стало:
- Отдельные страницы без скроллинга
- Только одна секция видна в каждый момент
- Переключение через навигацию
- История браузера (back/forward работает)

---

## Структура страниц

### 1. Обзор (Overview) - `/` или `#overview`
**Содержимое:**
- Hero секция с информацией о школе
- Статистика (инциденты, задачи, замены)
- Последние инциденты, задачи и замены

**Особенности:**
- Главная страница (по умолчанию)
- Сводная информация
- Быстрый доступ к ключевым метрикам

### 2. Расписание (Schedule) - `#schedule`
**Содержимое:**
- Выбор класса
- Таблица расписания

**Особенности:**
- Полноэкранная карточка
- Удобный просмотр расписания

### 3. Задачи (Tasks) - `#tasks`
**Содержимое:**
- Textarea для ввода задач
- Кнопка разбора задач
- AI обработка

**Особенности:**
- Фокус на постановке задач
- Без отвлекающих элементов

### 4. Замены (Substitutions) - `#substitutions`
**Содержимое:**
- Выбор учителя
- Выбор даты
- Варианты замены
- Применение замены

**Особенности:**
- Пошаговый процесс
- Полноэкранный режим

### 5. Приказы (Orders) - `#orders`
**Содержимое:**
- Textarea для приказа
- Кнопка упрощения
- Результат упрощения

**Особенности:**
- RAG модуль
- Большое пространство для текста

### 6. Чат (Chat) - `#chat`
**Содержимое:**
- Список контактов
- Окно чата
- Поле ввода сообщения

**Особенности:**
- Полноэкранный чат
- Высота адаптируется к экрану

## Технические детали

### CSS
```css
.page {
  display: none;
  animation: fadeIn 0.3s ease;
}

.page.active {
  display: block;
}

@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

### JavaScript навигация
```javascript
function showPage(pageId) {
  // Скрыть все страницы
  document.querySelectorAll('.page').forEach(page => {
    page.classList.remove('active');
  });
  
  // Показать выбранную
  document.getElementById(pageId).classList.add('active');
  
  // Обновить навигацию
  // Обновить URL
}
```

### История браузера
- Используется `history.pushState()` для обновления URL
- Обработчик `hashchange` для кнопок назад/вперёд
- URL вида: `http://localhost:3000/#tasks`

## Навигация

### Клик по вкладке:
1. Скрывает все страницы
2. Показывает выбранную
3. Обновляет активную вкладку
4. Обновляет URL (без перезагрузки)

### Кнопки браузера:
1. Отслеживается событие `hashchange`
2. Извлекается ID страницы из URL
3. Показывается соответствующая страница

### Прямая ссылка:
1. При загрузке читается hash из URL
2. Показывается соответствующая страница
3. Если hash пустой — показывается Overview
