# 🎤 Обновление: Голосовой ввод задач

## Новая функция

### Voice-to-Task реализован!
Теперь директор может надиктовывать задачи голосом, как требует ТЗ.

---

## Как работает

### 1. Нажмите кнопку "🎤 Голосовой ввод"
- Браузер запросит разрешение на микрофон
- Кнопка станет зелёной с анимацией пульса
- Появится статус "🎤 Говорите..."

### 2. Надиктуйте задачи
```
"Мы делаем хакатон на следующей неделе. 
Айгерим, подготовь актовый зал. 
Назкен, закажи воду и бейджи."
```

### 3. Остановите запись
- Нажмите "⏹️ Остановить" или просто перестаньте говорить
- Текст появится в поле ввода
- Статус покажет: "✅ Распознано: [ваш текст]"

### 4. Обработайте как обычно
- Нажмите "🚀 Разобрать и поставить задачи"
- Система распарсит голосовой текст
- Создаст задачи для исполнителей

## Технические детали

### Web Speech API
```javascript
const SpeechRecognition = window.SpeechRecognition || window.webkitSpeechRecognition;
recognition = new SpeechRecognition();

recognition.continuous = false;  // Одна фраза за раз
recognition.interimResults = false;  // Только финальный результат
recognition.lang = 'ru-RU';  // Русский язык
```

### Поддержка браузеров
- ✅ Chrome/Edge (полная поддержка)
- ✅ Safari (частичная поддержка)
- ❌ Firefox (не поддерживается)
- ✅ Мобильные браузеры (Android/iOS)

### Обработка ошибок
```javascript
recognition.onerror = (event) => {
  switch(event.error) {
    case 'no-speech': 'Речь не обнаружена'
    case 'audio-capture': 'Микрофон недоступен'
    case 'not-allowed': 'Доступ запрещён'
  }
}
```

## Ограничения

### Браузеры
- Firefox не поддерживает Web Speech API
- Старые браузеры могут не работать
- Требуется HTTPS для продакшена

### Качество распознавания
- Зависит от качества микрофона
- Фоновый шум может мешать
- Акцент может влиять на точность

### Языки
- Настроен на русский (ru-RU)
- Можно добавить другие языки
- Смешанные языки могут работать хуже
