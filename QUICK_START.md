# 🚀 Быстрый старт

## За 2 минуты до демо

### 1. Остановите старые процессы
```bash
pkill -9 node
```

### 2. Запустите сервер
```bash
npm run dev
```

### 3. Откройте браузер
```
http://localhost:3000
```

### 4. Проверьте что всё работает
- ✅ Дашборд загрузился
- ✅ Градиентный дизайн отображается
- ✅ Кнопки кликабельны

---

## Ответы на вопросы жюри

### "Почему не все учителя зарегистрированы?"
> Это демо-окружение. В продакшене будет обязательная регистрация.

### "Как обрабатываются конфликты расписания?"
> Алгоритм проверяет занятость учителя и предлагает только свободных.

### "Что если API недоступен?"
> Есть fallback-парсер для задач. Для приказов — повтор запроса.

### "Как масштабируется?"
> Текущая версия — MVP. Для продакшена: PostgreSQL + Redis + очереди.

### "Почему JSON вместо БД?"
> Для MVP достаточно. Миграция на PostgreSQL — 1 день работы.

## Если что-то сломалось

### Сервер не запускается (EADDRINUSE)
```bash
pkill -9 node
npm run dev
```

### Дашборд не загружается
```bash
# Проверьте что сервер запущен
curl http://localhost:3000/api/health
```

### Telegram бот не отвечает
```bash
# Проверьте .env
cat .env | grep TELEGRAM_BOT_TOKEN
```

### Gemini не работает
```bash
# Проверьте .env
cat .env | grep GEMINI_API_KEY
```
