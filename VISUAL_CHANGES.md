## Цветовая схема

### Кнопки
```css
/* Обычное состояние */
background: #111
color: white

/* Hover */
background: #2d2d2d

/* Disabled (новое) */
background: #999
opacity: 0.6
cursor: not-allowed
```

### Результат упрощения приказа (новое)
```css
background: #f0f8ff (светло-голубой)
border-left: 4px solid #0066cc (синий)
border-radius: 8px
padding: 16px
line-height: 1.6
```

### Статусы
- ✅ Успех: зелёная галочка
- ⏳ Загрузка: часы
- ❌ Ошибка: красный крестик

## Анимации и переходы

### Кнопки
```css
transition: all 0.2s ease;
```

При нажатии:
1. Мгновенно: `disabled = true`
2. Мгновенно: текст меняется на "⏳ ..."
3. Плавно (0.2s): цвет меняется на серый
4. После ответа сервера: возврат в исходное состояние

## Доступность (A11y)

### Улучшения:
- ✅ Кнопки имеют `disabled` атрибут (screen readers объявляют состояние)
- ✅ Статусы операций читаются screen readers
- ✅ Цветовой контраст соответствует WCAG AA:
  - Кнопки: белый на чёрном (21:1)
  - Результат приказа: тёмный текст на светлом фоне (>7:1)

### Клавиатурная навигация:
- Tab: переход между кнопками
- Enter/Space: активация кнопки
- Заблокированные кнопки пропускаются при Tab
