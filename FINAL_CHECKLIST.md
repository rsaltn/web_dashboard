## Известные ограничения

### Не критично для демо:
- Только один класс в расписании (10AS)
- Мало учителей зарегистрировано в боте
- JSON вместо БД (race conditions возможны)
- Нет голосового ввода (только текст)
- Нет автосвода в 09:00 (только ручной)

### Можно объяснить жюри:
- "Это MVP, в продакшене будет PostgreSQL"
- "Голосовой ввод можно добавить через Web Speech API"
- "Cron для автосвода настраивается за 5 минут"
