# 🧭 Руководство по навигации

## Как работает

### Структура
Сайт теперь состоит из 6 отдельных страниц:

1. **Обзор** (`#overview`) - главная страница со статистикой
2. **Расписание** (`#schedule`) - просмотр расписания классов
3. **Задачи** (`#tasks`) - постановка задач сотрудникам
4. **Замены** (`#substitutions`) - замена заболевших учителей
5. **Приказы** (`#orders`) - упрощение приказов через AI
6. **Чат** (`#chat`) - общение с учителями

### Переключение страниц
- Кликните на вкладку в навигации
- Страница переключится без перезагрузки
- URL обновится автоматически
- Кнопки назад/вперёд браузера работают

### Прямые ссылки
Можно открыть конкретную страницу:
- `http://localhost:3000/` - Обзор
- `http://localhost:3000/#schedule` - Расписание
- `http://localhost:3000/#tasks` - Задачи
- `http://localhost:3000/#substitutions` - Замены
- `http://localhost:3000/#orders` - Приказы
- `http://localhost:3000/#chat` - Чат

---

## Преимущества

✅ **Нет скроллинга** - весь контент виден сразу
✅ **Быстрая навигация** - один клик для переключения
✅ **Фокус на задаче** - только нужная информация
✅ **История браузера** - работают кнопки назад/вперёд
✅ **Мобильная версия** - удобно на телефоне
